Outcome types

● Cleared
Full-res original delivered. All rights checks passed for this user. Standard primary download button.
● Watermarked
Rights mismatch but user is internal. Watermarked working copy delivered with visible "Not cleared for use" overlay. Logged.
● Blocked
Hard mismatch — expired, wrong audience, or external user with no entitlement. Download disabled with click-to-reveal reason.
● Request Access
Blocked but recoverable. User can submit a request that goes to the rights owner for approval. Only path to a higher entitlement.
