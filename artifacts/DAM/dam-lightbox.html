Annotations
Filename leads
Now the dominant element in the header. The user's primary question — "what am I looking at?" — gets answered first, without contest from other elements.
Hover demo
Right nav arrow and middle zoom button shown in hover state. Ghost buttons gain a subtle background and darker text on hover; zoom buttons gain a faint white overlay. Full state range in Section 4.
Ghost arrows
No borders, smaller (28×28), tight against the filename. Still discoverable, still semantically linked to "this file" — but subordinate to the filename, as they should be.
Counter out of header
"12 of 2,056" relocated to the right of the footer. Still findable, still useful when needed, but no longer crowding the primary identity.
Kbd hints compacted
Footer labels shortened: "Asset" not "Previous / next asset"; "Artboard" not "Previous / next artboard". The arrow icons carry the meaning; the word just labels what they navigate.
Single artboard counter
Removed the duplicate "Artboard 1 of 6" from the footer. Stage already shows it bottom-right. Footer carries only the asset-set counter, no overlap.

Back-to-source chip — similarity navigation
Appears in the header after a user navigates to another asset via the Similar tab. Single click returns to the original source asset. Clears on click, on lightbox close, or on a fresh session. Not stack-based — single-step return only.

Annotations
Back-to-source chip
Pill with primary-tinted background, return icon, and the source filename. Sits between the nav arrows and the current filename — visually marks the contextual flow ("you arrived here from there").
Single-step return
Clicking returns to the original source asset only — no stacked history. If the user navigates from A → B → C via Similar, the chip on C still says "Back to A" (not B). Per the SCR-113 spec — Phase 1 keeps this simple.
Chip clears on
Click (returns to source), lightbox close, or fresh session start. Browsing to a fully unrelated asset via the result-set arrows also clears it.
Nav arrows context shifts
When in a similarity flow, ← / → navigate the similar-results set (12 items), not the library result set. Footer shows "2 of 12 similar" instead of the library counter. Hint label changes to "Similar".
Chip filename truncates
If the source filename is long, the chip truncates with ellipsis (max ~200px). Tooltip on hover shows the full filename for confirmation.
Filename style preserved
Current asset filename still in monospace bold to mark it as the primary identity. Source filename in the chip uses smaller monospace — same family for visual cohesion, lower weight to keep it subordinate to the current asset.
