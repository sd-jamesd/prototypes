OCR match — flag system

One shared match-reason chip across the journey — quick search → result card → lightbox — so a text hit reads as the same object everywhere. Accent + Lucide scan-search icon, on Scale --primary tokens. Copy branches by source: text in document (PDF/DOCX) or text in image (PNG/JPG/TIFF OCR). It is not a .status-badge (no status dot, not a workflow state) and not a .filetype-badge (not solid-fill). On the list surfaces the chip carries only the reason; the matched term in context, the match count and (for documents) the page belong to the lightbox banner. The banner appears for both documents and images — anywhere the matched text is hard to find in the preview. Documents show the page reference; images omit it (no pages) and rely on the context snippet to locate the match.
